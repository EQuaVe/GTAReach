Detailed Description

template<class NODE, class EDGE, class NODE_HASH, class NODE_EQUAL>
class tchecker::graph::reachability::graph_t< NODE, EDGE, NODE_HASH, NODE_EQUAL >

Graph that allocates and stores nodes and edges in a reachability graph.

Template Parameters
NODE: type of nodes
EDGE: type of edges
NODE_HASH: hash function on nodes
NODE_EQUAL: equality predicate on nodes
Note
this graph allocates nodes of type tchecker::graph::reachability::node_t<NODE, EDGE> and edges of type tchecker::graph::reachability::edge_t<NODE, EDGE>

Detailed Description

template<class TS, class GRAPH>
class tchecker::algorithms::reach::algorithm_t< TS, GRAPH >

Reachability algorithm.

Template Parameters
TS: type of transition system, should derive from tchecker::ts::ts_t
GRAPH: type of graph, should derive from tchecker::graph::reachability_graph_t, and nodes of type GRAPH::shared_node_t should have a method state_ptr() that yields a pointer to the corresponding state in TS

Build a reachability graph of a transition system from its initial states.

Parameters
ts: a transition system
graph: a graph
labels: accepting labels
policy: waiting list policy
Postcondition
graph is built from a traversal of ts starting from its initial states, until a state that satisfies labels is reached (if any). A node is created for each reachable state in ts, and an edge is created for each transition in ts. The order in which the nodes of ts are visited depends on policy.
Returns
statistics on the run
Note
if labels is empty, graph is the full reachability graph of ts

Parameters

Postcondition

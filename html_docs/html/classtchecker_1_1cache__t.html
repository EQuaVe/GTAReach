Detailed Description

template<class T, class HASH, class EQUAL>
class tchecker::cache_t< T, HASH, EQUAL >

Cache of shared objects.

Parameters
T: type of objects, should be tchecker::make_shared_t<>
HASH: type of hash function on T, should be default constructible
EQUAL: type of equality predicate on T, should be default constructible

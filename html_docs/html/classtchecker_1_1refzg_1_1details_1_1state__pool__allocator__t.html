Detailed Description

template<class STATE>
class tchecker::refzg::details::state_pool_allocator_t< STATE >

Pool allocator for states of zone graphs with reference clocks that can be extended to allocate more complex states.

Template Parameters
STATE: type of states, should inherit from tchecker::refzg::state_t and should be a tchecker::make_shared object

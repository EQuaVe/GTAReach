inlinevirtual

Visitor.

Postcondition
If expr is a lower-bound guard (x>c, x>=c, x==c), then a constraint on clock lower bounds has been added to _solver for every clock x (x could be an array) w.r.t. bound c (using method add_lower_bound_guard). If expr is an upper-bound guard (x<c, x<=c, x==c), then a constraint on clock upper bounds has been added to _solver for every clock x (x could be an array) w.r.t. bound c (using method add_upper_bound_guard).

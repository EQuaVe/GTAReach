Resize the clock bounds mapsa.

Parameters
loc_nb: number of locations
clock_nb: number of clocks
Note
all location ID in [0..loc_nb) are assumed to be valid and all clock ID in [0..clock_nb) are assumed to be valid
Postcondition
all clock bounds has been cleared and resized to loc_nb locations and clock_nb clocks

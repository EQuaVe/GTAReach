Detailed Description

template<class R, class SUBR>
class tchecker::join_iterator_t< R, SUBR >

Join iterator over ranges. join_iterator_t makes a range of ranges ((x11,...,x1N),...,(xK1,...,xKM)) appear as the flat range (x11,...,x1N,...,xK1,...,xKM), skipping empty sub-ranges.

Template Parameters
R: type of range of sub-ranges, should be a tchecker::range_t
SUBR: type of sub-range, should be a tchecker::range_t

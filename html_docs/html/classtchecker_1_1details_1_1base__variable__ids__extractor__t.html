Detailed Description

Extract base variable IDs in a typed lvalue expression (tchecker::typed_var_expression_t, tchecker::typed_bounded_var_expression_t or tchecker::typed_array_expression_t)

Note
base variables are simple variables and base names of array expression (i.e. x in x[e])
Exceptions
std::invalid_argument: if the expression is not an lvalue expression

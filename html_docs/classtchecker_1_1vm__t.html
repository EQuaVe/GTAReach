Bytecode interpreter.

Parameters
bytecode: tchecker bytecode
intval: valuation of bounded integer variables
clkconstr: container of clock constraints
clkreset: container of clock resets
Precondition
bytecode is null-terminated (i.e. VM_RET). Variables identifiers in bytecode are less than intval.size() (checked by assertion)
Returns
value computed by the last instruction in bytecode
Postcondition
bytecode has been executed: intval has been updated, clock constraints have been pushed into clkconstr, and clock resets have been pushed into clkreset
Exceptions
std::runtime_error: if bytecode interpretation fails
std::out_of_range: if out-of-bound array access
Note
Evaluating a bytecode from an expression returns 0 for false and any other value for true
Evaluating a bytecode from a statement always returns 1. It throws an exception if evaluation failed.

Detailed Description

template<class TRANSITION>
class tchecker::syncprod::details::transition_pool_allocator_t< TRANSITION >

Pool allocator for transitions of synchronized product of transition systems that can be extended to allocate more complex transitions.

Template Parameters
TRANSITION: type of transitions, should inherit from tchecker::syncprod::transition_t and should be a tchecker::make_shared object

Detailed Description

template<class T>
class tchecker::singleton_pool_t< T >

Singleton pool allocator.

Template Parameters
T: type of allocated object
Note
A singleton pool allocates a unique object. Each call to construct() returns the same address. The object at this address is reset at each construction.

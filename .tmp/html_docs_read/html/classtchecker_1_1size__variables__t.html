Detailed Description

template<class ID, class INFO, class INDEX>
class tchecker::size_variables_t< ID, INFO, INDEX >

Definition of variables with a size (arrays)

Template Parameters
ID: type of variable identifiers
INFO: type of variable informations, should derive from tchecker::size_info_t
INDEX: type of variable index, should derive from tchecker::index_t
Note
Variables identifiers take into account the size of the variables: when a variable is declared all identifiers needed by the variable are reserved

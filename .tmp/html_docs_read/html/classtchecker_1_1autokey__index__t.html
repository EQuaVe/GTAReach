Detailed Description

template<class KEY, class T>
class tchecker::autokey_index_t< KEY, T >

Stores an bijective bimap KEY <-> T where keys are generated automatically.

Template Parameters
KEY: type of keys, must be an unsigned integer type
T: type of values
Note
see boost::container::flat_map for requirements on T

Detailed Description

Clock constraint.

Note
Represents a constraint x_i - x_j # k where i and j are IDs of clock variables, # is either < or <=, and k is an integer. A constraint is diagonal if both i and j are distinct from tchecker::REFCLOCK_ID. If one of i and j is equal to tchecker::REFCLOCK_ID, the constraint is simple. Hence, x_i - x_j # k with j = tchecker::REFCLOCK_ID encodes the constraint x_i # k. And similarly, the constraints encodes -x_j # k when i = tchecker::REFCLOCK_ID.

Detailed Description

template<class R>
class tchecker::cartesian_iterator_t< R >::values_iterator_t

Iterator over a tuple of values in the cartesian product.

Note
Makes a vector of iterator appear as the sequence of values pointed by those iterators. Could be achieved using boost::transform_iterator, but has typing issues

◆ process_reference_clocks()

tchecker::reference_clock_variables_t tchecker::process_reference_clocks ( tchecker::variable_access_map_t const &  vaccess_map,
tchecker::flat_clock_variables_t const &  flat_clocks,
tchecker::process_id_t  proc_count 
)

Build refeence clock variables from a variable access map.

Parameters
vaccess_map: a variable access map
flat_clocks: flat clock variables
proc_count: number of processes
Precondition
proc_count > 0 all process IDs in vaccess_map should belong to [0,proc_count)
Returns
reference clock variables with proc_count reference clocks, along with all clocks from flat_clocks. Each clock is mapped to the reference clock of the process that accesses the clock according to vaccess_map
Exceptions
std::invalid_argument: if proc_count == 0
std::invalid_argument: if the number of clocks (i.e. proc_count reference clocks + flat_clocks size) exceeds the number of representable clock identifiers

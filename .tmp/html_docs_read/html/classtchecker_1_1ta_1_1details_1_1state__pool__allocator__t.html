Detailed Description

template<class STATE>
class tchecker::ta::details::state_pool_allocator_t< STATE >

Pool allocator for states of timed automata that can be extended to allocate more complex states.

Template Parameters
STATE: type of states, should inherit from tchecker::ta::state_t and should be a tchecker::make_shared object

Detailed Description

Vector of edges.

Note
The vector of edges is implemented as an array of edge identifiers. For each process P, the array either contains the identifier of the edge taken by P, or tchecker::NO_EDGE if P is not involved in this vector of edges. Hence, direct accesses to the array may give value tchecker::NO_EDGE which is not a valid edge identifier. On the other hand, iterators returned by methods begin() and end() range over valid edge identifiers.

Detailed Description

template<class ID, class INFO, class INDEX>
class tchecker::flat_variables_t< ID, INFO, INDEX >

Definition of flat variables (i.e. variables of size 1)

Template Parameters
ID: type of variable identifiers
INFO: type of variable informations, should derive from tchecker::size_info_t
INDEX: type of variable index, should derive from tchecker::index_t

tchecker::flat_variables_t< ID, INFO, INDEX >::flat_variables_t ( tchecker::variables_t< ID, INFO, INDEX > const &  v)
inline

Constructor.

Parameters
v: variables
Precondition
variable identifiers in v do not overlap
Postcondition
this has been built from flattening every variable in v. Each variable in v has been transformed in v's size consecutive variables of size 1 with same information as in v (except for the size). The variable indices start from the index of the corresponding base variable in v
Exceptions
std::invalid_argument: if variables in v overlap

◆ dbm() [1/3]

const tchecker::dbm::db_t * tchecker::zg::zone_t::dbm ( )

Accessor.

Returns
internal DBM of size dim()*dim()
Note
Modifications to the returned DBM should ensure tightness or emptiness of the zone, following the convention defined in file tchecker/dbm/dbm.hh. It is thus strongly suggested to use the function defined in that file to modify the returned DBM. Otherwise, the methods may not be accurate over this zone.

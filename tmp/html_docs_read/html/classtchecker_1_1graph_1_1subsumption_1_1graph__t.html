Detailed Description

Graph that allocates and stores nodes and edges in a subsumption graph. This graph allows to find covered nodes and covering nodes w.r.t. a covering predicate. It stores two kind of edges: actual edges and subsumption edges.

Template Parameters
NODE: type of nodes
EDGE: type of edges
NODE_HASH: hash function on nodes, should be callable with a parameter of type NODE const &, and return the hash value of the node
NODE_LE: covering predicate on nodes, should be callable with two parameters of type NODE const &, and return true is the first node is covered by the second one, false otherwise
Note
this graph allocates nodes of type tchecker::graph::subsumption::node_t<NODE, EDGE> and edges of type tchecker::graph::subsumption::edge_t<NODE, EDGE>

Detailed Description

Graph with node covering.

Template Parameters
NODE_PTR: type of pointer to node, should be shared pointer, either std::shared_ptr<> or tchecker::intrusive_shared_ptr_t<> of some node type
NODE_HASH: type of hash function on node pointers. Should be callable with a NODE_PTR argument and return a hash code for the pointed node
NODE_LE: less-than-or-equal predicate on nodes. Should be callable with two NODE_PTR argument and return true if the first one is smaller than the second one, and false otherwise
Note
This graph allows to check if there is a node in the graph that covers some given node. Nodes are compared using NODE_LE. Only the nodes with the same hash value w.r.t. NODE_HASH are compared

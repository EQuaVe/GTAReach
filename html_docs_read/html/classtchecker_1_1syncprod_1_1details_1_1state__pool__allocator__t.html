Detailed Description

template<class STATE>
class tchecker::syncprod::details::state_pool_allocator_t< STATE >

Pool allocator for states of synchronized product of transition systems that can be extended to allocate more complex states.

Template Parameters
STATE: type of states, should inherit from tchecker::syncprod::state_t and should be a tchecker::make_shared object

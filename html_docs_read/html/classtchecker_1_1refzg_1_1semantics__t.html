Compute initial zone with reference clocks.

Parameters
rdbm: a DBM
r: reference clocks for rdbm
delay_allowed: set of reference clocks allowed to delay
invariant: invariant
spread: reference clocks spread
Precondition
rdbm is not nullptr (checked by assertion). rdbm is a r.size()*r.size() array of difference bounds. rdbm is tight and consistent. the size of delay_allowed is the number of reference clocks in r
Postcondition
dbm is the initial zone w.r.t. delay_allowed, invariant and spread
Returns
STATE_OK if the resulting DBM is not empty, other values if the resulting DBM is empty (see details in implementations)

Compute next zone with reference clocks.

Parameters
rdbm: a DBM
r: reference clocks for rdbm
src_delay_allowed: set of reference clocks allowed to delay in source state
src_invariant: invariant in source state
sync_ref_clocks: set of reference clocks to synchronize
guard: transition guard
clkreset: transition reset
tgt_delay_allowed: set of reference clocks allowed to delay in target state
tgt_invariant: invariant in target state
Precondition
rdbm is not nullptr (checked by assertion). rdbm is a r.size()*r.size() array of difference bounds. rdbm is tight and consistent. the size of src_delay_allowed is the number of reference clocks in r. the size of sync_ref_clocks is the number of reference clocks in r. the size of tgt_delay_allowed is the number of reference clocks in r
Postcondition
rdbm has been updated to its strongest postcondition w.r.t. src_delay_allowed, src_invariant, sync_ref_clocks, guard, clkreset, tgt_delay_allowed and tgt_invariant
Returns
STATE_OK if the resulting DBM is not empty, other values if the resulting DBM is empty (see details in implementations)

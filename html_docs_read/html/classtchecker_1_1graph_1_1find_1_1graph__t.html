Detailed Description

template<class NODE_PTR, class HASH, class EQUAL>
class tchecker::graph::find::graph_t< NODE_PTR, HASH, EQUAL >

Graph with node finding.

Template Parameters
NODE_PTR: type of pointer to node
HASH: hash function on NODE_PTR (see std::hash)
EQUAL: equality function on NODE_PTR (see std::equal_to)
Note
this graph implementation stores nodes and answers find queries. It does not store edges
each node has a unique instance in this graph w.r.t. EQUAL

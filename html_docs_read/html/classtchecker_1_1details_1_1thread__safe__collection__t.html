Detailed Description

template<class T, class EQUAL>
class tchecker::details::thread_safe_collection_t< T, EQUAL >

Collection of shared objects with thread-safe access.

Parameters
T: type of object, should be tchecker::make_shared_t<>
EQUAL: equality predicate on T values, should be default constructible
